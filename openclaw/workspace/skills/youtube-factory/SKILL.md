# YouTube Factory Skill

Automatisierte YouTube Video Produktion in 10 Schritten.
Siehe YOUTUBE_PIPELINE_SOP.md fuer vollstaendige Dokumentation.

## Befehle
- `/youtube-new [topic]` — Neues Video starten (alle 10 Schritte)
- `/youtube-script [topic]` — Nur Script generieren
- `/youtube-shorts [video]` — 3 Shorts aus Video schneiden
- `/youtube-status` — Status aller Videos in Pipeline

## Quick Pipeline
```
Topic → Script → Thumbnail → Voice → Footage → Captions → Upload → Shorts → Track → Review
```

## Tools (alle kostenlos)
- Piper TTS (Voiceover)
- Whisper (Untertitel)
- FFmpeg (Video-Editing)
- Pillow (Thumbnails)
- yt-dlp (Research)
- Pexels API (Stock Footage)
