# LinkedIn Integration

Integrate your LinkedIn professional network with CORE.

## Features

- **Activity Tracking**: Syncs your posts and shared content.
- **Profile Data**: Fetches basic profile information.
- **Posting**: Create new posts directly from CORE.

## Setup

1.  Create a LinkedIn App in the [LinkedIn Developer Portal](https://www.linkedin.com/developers/apps).
2.  Enable "Sign In with LinkedIn" and "Share on LinkedIn" products.
3.  Get your Client ID and Client Secret.
4.  Configure the integration in CORE with your credentials.
