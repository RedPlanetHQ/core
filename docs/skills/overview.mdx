---
title: "Skills"
description: "Tell CORE what matters to you once — every task reads from the same definition."
---

Skills are personal policies — the things that are important to *you*, written down once. They aren't runnable tasks. They're the lenses, preferences, and definitions that tasks apply.

A skill answers a question about you: *what counts as important email?* *how do I look at an account?* *what does healthy engineering velocity look like for me?* Once written, every surface — inbound triage, agent flows, daily briefs, scheduled reports — uses the same answer.

To create one: **Dashboard → Agent → Skills → New Skill**. Add a title, the policy, and a short description so CORE knows when to load it.
