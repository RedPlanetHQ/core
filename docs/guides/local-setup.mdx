## Getting Started

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/install/)
- Node.js 18+ & pnpm

### Setup
1. Fork this repository
2. Clone your fork locally and install dependencies
   ```bash
   git clone https://github.com/RedPlanetHQ/core.git
   cd core
   pnpm install
   ```
3. Copy ```.env.example``` to ```.env``` and add your __OPENAI_API_KEY__.

5. Start docker services
    ```bash
    cd hosting/docker
    docker-compose up -d core-postgres core-redis core-neo4j
    cd ../..
    ```
6. Start the server
    ```bash
    pnpm dev
    ```
The application will be running at ```http//:localhost:3033```
