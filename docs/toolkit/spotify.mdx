## Overview

Integrate Spotify to access your personal music data and the Spotify catalog directly from CORE. Access 14 tools covering your currently playing track, recently played history, saved library, playlists, and full catalog search.

## Setup Guide

- `Authentication Type`: OAuth 2.0 (Spotify)

## How to Connect

1. Navigate to [CORE Dashboard](https://app.getcore.me) → **Integrations**
2. Click on the **Spotify** card
3. Click **Connect** to authorize CORE
4. You'll be redirected to Spotify to authorize the application
5. Grant the required permissions
6. You'll be redirected back to CORE once connected

## Scopes

- `user-read-playback-state` - Read current playback state and device info
- `user-read-currently-playing` - Read the currently playing track
- `user-read-recently-played` - Read recently played tracks
- `user-library-read` - Access saved tracks and albums
- `playlist-read-private` - Read private playlists
- `playlist-read-collaborative` - Read collaborative playlists
- `user-read-email` - Read user email (used as account identifier)
- `user-read-private` - Read user profile and subscription details

## Available MCP Tools

<Tabs>
  <Tab title="User & Playback">

| Tool Name | Description |
|-----------|-------------|

| `get_playback_state` | Get full playback state including device, progress, shuffle, and repeat |

## Use Cases

<AccordionGroup>
  <Accordion title="What's Playing Right Now" icon="music">
    **Scenario**: Check the currently playing track and full playback context

    ```
    "What am I listening to on Spotify right now?"
    ```

    The agent will use `get_currently_playing` to return the track name, artist, album, progress, and whether it's playing or paused.
  </Accordion>

  <Accordion title="Recently Played History" icon="clock-rotate-left">
    **Scenario**: Review your recent listening activity

    ```
    "What were the last 10 songs I listened to on Spotify?"
    ```

    The agent will use `get_recently_played` and return each track with its played-at timestamp.
  </Accordion>

  <Accordion title="Browse Your Library" icon="heart">
    **Scenario**: Explore your saved tracks and playlists

    ```
    "Show me my saved tracks on Spotify"
    ```

    The agent will use `get_saved_tracks` to list your liked songs with album and artist info, or `get_user_playlists` to list all your playlists.
  </Accordion>

  <Accordion title="Catalog Search" icon="magnifying-glass">
    **Scenario**: Find tracks, artists, or albums from the Spotify catalog

    ```
    "Find the Spotify track ID for 'Bohemian Rhapsody' by Queen"
    ```

    The agent will use `search_tracks` with the query and return matching tracks with IDs, duration, and album info.
  </Accordion>

  <Accordion title="Artist Discovery" icon="star">
    **Scenario**: Look up an artist's top tracks

    ```
    "What are Taylor Swift's top tracks on Spotify right now?"
    ```

    The agent will use `search_artists` to find Taylor Swift's artist ID, then `get_artist_top_tracks` to retrieve her current top tracks.
  </Accordion>

  <Accordion title="Album Details" icon="record-vinyl">
    **Scenario**: Explore an album's track listing

    ```
    "List all the tracks on Daft Punk's Random Access Memories"
    ```

    The agent will use `search_albums` to find the album, then `get_album_tracks` to list all tracks with their durations.
  </Accordion>
</AccordionGroup>
