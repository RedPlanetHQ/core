## Overview

Integrate Spotify to search and retrieve data from the Spotify catalog directly from CORE. Access 8 tools for tracks, artists, and albums using your Spotify Developer app credentials — no user OAuth flow required.

## Setup Guide

- `Authentication Type`: API Key (Client Credentials — Client ID + Client Secret)

## How to Connect

1. Go to the [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
2. Create a new app (or use an existing one)
3. Copy the **Client ID** and **Client Secret** from the app settings
4. Navigate to [CORE Dashboard](https://app.getcore.me) → **Integrations**
5. Click on the **Spotify** card
6. Enter your Client ID and Client Secret
7. Click **Connect** — CORE validates the credentials and saves the integration

> **Note**: Client Credentials auth provides access to the **public Spotify catalog only** (tracks, artists, albums, playlists). It does not access user-specific data such as listening history or playlists.

## Available MCP Tools

| Tool Name | Description |
|-----------|-------------|
| `search_tracks` | Search for tracks in the Spotify catalog |
| `get_track` | Get details of a specific track by Spotify track ID |
| `search_artists` | Search for artists in the Spotify catalog |
| `get_artist` | Get details of a specific artist by Spotify artist ID |
| `get_artist_top_tracks` | Get an artist's top tracks in a given market |
| `search_albums` | Search for albums in the Spotify catalog |
| `get_album` | Get details of a specific album by Spotify album ID |
| `get_album_tracks` | Get the tracks of a specific album |

## Use Cases

### Search for a track

```
"Find the Spotify track ID for 'Bohemian Rhapsody' by Queen"
```

The agent will use `search_tracks` with the query `"Bohemian Rhapsody artist:Queen"` and return matching tracks with IDs, duration, and album info.

### Look up an artist's top tracks

```
"What are Taylor Swift's top tracks on Spotify right now?"
```

The agent will use `search_artists` to find Taylor Swift's artist ID, then `get_artist_top_tracks` to retrieve her current top tracks.

### Get album details

```
"List all the tracks on Daft Punk's Random Access Memories album"
```

The agent will use `search_albums` to find the album, then `get_album_tracks` to list all tracks with their durations.
