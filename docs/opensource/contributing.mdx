## Contributing

CORE is open-source and welcomes community contributions.

**Ways to contribute:**
- Report bugs or suggest features on [GitHub](https://github.com/redplanethq/core)
- **Build integrations** (Jira, ClickUp, Trello, etc.) - [See guide](/opensource/building-integrations)
- Write documentation
- Review code
- Share CORE with others
- Give feedback

Your contributions make CORE better for everyone.
