CORE's MCP endpoint supports flexible configuration through query parameters.

## Base URL

```
https://app.getcore.me/api/v1/mcp
```

## Parameters

### Required

**source** - Identifies your connection in the dashboard
```
?source=Claude
?source=Cursor
?source=VSCode
```

### Optional

**integrations** - Specify which integrations to include
```
&integrations=github,linear
&integrations=linear
```

**no_integrations** - Disable all integrations (CORE tools only)
```
&no_integrations=true
```

## Examples

**All integrations:**
```
https://app.getcore.me/api/v1/mcp?source=Claude
```

**Specific integrations:**
```
https://app.getcore.me/api/v1/mcp?source=Cursor&integrations=github,linear
```

**CORE tools only:**
```
https://app.getcore.me/api/v1/mcp?source=VSCode&no_integrations=true
```
