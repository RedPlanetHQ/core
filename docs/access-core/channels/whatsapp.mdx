## Overview

Access CORE's Meta Agent directly through WhatsApp. Get instant answers, take actions across your apps, and search your memory - all from your phone. Perfect for on-the-go productivity.

<Info>
  **Private Beta**: WhatsApp access is currently in private beta. [Join the waitlist](https://app.getcore.me/settings/channels) to get early access.
</Info>

## Getting Started

1. Go to [app.getcore.me/settings/channels](https://app.getcore.me/settings/channels)
2. Click **Enable WhatsApp** and join the waitlist
3. Once approved, scan the QR code to connect
4. Start messaging your Meta Agent on WhatsApp

## Tips

- **Keep it short**: WhatsApp works best with concise commands
- **Use voice**: Voice messages make it even faster (coming soon)
- **Set up shortcuts**: Create quick commands for frequent actions
- **Enable notifications**: Get real-time updates from your tools

## Privacy & Security

- End-to-end encrypted communication
- WhatsApp integration uses official Business API
- Messages processed in real-time, not stored on WhatsApp servers
- You control all data retention policies

## Limitations

- File uploads limited to 16MB
- Voice messages in beta (coming soon)
- Group chat support coming soon
